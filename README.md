# Teacher Dashboard & Student Onboarding

This repository contains the work done for the **Production Didactique** course.

## Contents

- **Onboarding.svelte** – Guided tour for students (3 steps: sidebar, courses grid, support button).
- **Teacher dashboard** – Class cards, create class modal, layout.
- **ClassCard.svelte** – Course card with header images and action links.
- **CreateClassModal.svelte** – Modal form for creating a new class.
- **Images** – Thematic headers for courses (math, physics, CS, English), generated with Gemini AI.

## Usage

These components are designed for the OpenTutorAI SvelteKit project. Place them in the appropriate directories (`src/lib/components/student/`, `src/routes/student/dashboard/`, etc.) to integrate.
